Front End Web Development

Front-end web development builds the visible, interactive parts of websites (user interface/experience) using core languages like HTML, CSS, and JavaScript, along with frameworks (React, Angular, Vue) and tools (Git) to create engaging, responsive, and accessible designs that users directly see and interact with, bridging graphic design and back-end functionality

HTML

HTML (Hypertext Markup Language) is the standard language for creating web pages, defining their structure and content using tags to format text, images, and links for display in web browsers.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and visual formatting of a document written in a markup language, most commonly HTML

JAVASCRIPT

JavaScript is a versatile, high-level programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is used to create dynamic and interactive experiences on websites and in other applications.
